# OverlayMusic v1.2.2 — Watchdog NPSMSvc + автообновление

## Что нового

- 🐕 **Watchdog для службы NPSMSvc.** У этой системной службы Windows (Now Playing Session Manager) есть давний баг — она иногда застревает в high-CPU loop (до 100% одного ядра, сотни МБ памяти) после некорректного отключения какого-то SMTC-клиента (часто это закрытая вкладка Edge). Лечится только рестартом службы. Теперь оверлей сам это делает: каждые 15 сек меряет CPU% процесса службы; если 2 проверки подряд видим > 30% одного ядра — рестартует через `Restart-Service` (работает потому что мы запускаемся от админа через scheduled task). После рестарта — cooldown 5 минут, чтобы не зацикливаться. В трей вылетает уведомление с измеренной нагрузкой.
- 🚀 **Автообновление с GitHub.** Через 5 сек после старта и потом раз в 6 часов оверлей дёргает GitHub Releases API. Если есть новее версия — в трее всплывает «Доступна v1.X.Y. Кликни «Установить обновление»» и в контекстном меню появляется одноимённый пункт. По клику — скачивает `OverlayMusic-Setup-X.Y.Z.exe` в `%TEMP%`, запускает с `/VERYSILENT /SUPPRESSMSGBOXES /NORESTART /FORCECLOSEAPPLICATIONS /RESTARTAPPLICATIONS` (Inno Setup умеет апгрейдить по AppId), наш процесс завершается. Без подтверждения ничего не делает — кликаешь когда удобно.

## Установка

### 🪟 Windows
- **`OverlayMusic-Setup-1.2.2.exe`** — установщик (Program Files + автозапуск без UAC через Scheduled Task)
- **`OverlayMusic.exe`** — портативный exe

> ⚠ Не подписан Authenticode → SmartScreen покажет предупреждение. Жми **«Подробнее» → «Выполнить в любом случае»**.

### 🍏 macOS (Apple Silicon — M1/M2/M3)
- **`OverlayMusic-arm64.dmg`** — перетащи в Applications. Watchdog NPSMSvc не нужен (это Windows-фича); автообновление пока тоже только в Windows-версии.
