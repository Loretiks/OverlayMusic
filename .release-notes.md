# OverlayMusic v1.2.0 — Плавный drag + анимация эквалайзера

## Что нового

- ⚡ **Перетаскивание окна теперь плавное.** Раньше при каждом mouse-move пересчитывалась размытая подложка через `SmoothTransformation` на 480×168 — отсюда лаги. Теперь подложка кешируется при смене обложки/размера, drag не дёргается.
- 🎵 **Анимация эквалайзера на обложке.** В правом нижнем углу обложки появились 4 столбика, которые прыгают пока играет музыка и замирают на паузе. На светлых обложках читаются благодаря тёмной pill-подложке.

## Установка

### 🪟 Windows
- **`OverlayMusic-Setup-1.2.0.exe`** — установщик (ставит в Program Files + автозапуск без UAC через Scheduled Task)
- **`OverlayMusic.exe`** — портативный exe

> ⚠ Не подписан Authenticode → SmartScreen покажет предупреждение. Жми **«Подробнее» → «Выполнить в любом случае»**.

### 🍏 macOS (Apple Silicon — M1/M2/M3)
- **`OverlayMusic-arm64.dmg`** — перетащи в Applications. Первый запуск: ПКМ → **Open**. Если не открывается: `xattr -dr com.apple.quarantine /Applications/OverlayMusic.app`
