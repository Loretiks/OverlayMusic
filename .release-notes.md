# OverlayMusic v1.0.0 — Windows release

Первый публичный релиз. macOS-сборка появится в **v1.1.0** после тестирования на M-серии.

## Установка

### Рекомендуется: установщик (47 МБ)
Скачай **`OverlayMusic-Setup-1.0.0.exe`** ниже и запусти.
- Требует прав администратора (UAC-промпт)
- Ставит в `C:\Program Files\OverlayMusic\`
- По умолчанию активирует **автозапуск через Scheduled Task** — оверлей будет запускаться при каждом входе в Windows автоматически и **без UAC-промпта** (через `schtasks /RL HIGHEST /SC ONLOGON`)
- Удаляется через «Установка и удаление программ» (заодно убирает Scheduled Task)

### Альтернатива: standalone (47 МБ)
Скачай **`OverlayMusic.exe`** для портативного использования без установки.
- Запускать с правами админа (если нужна работа поверх игр от админа)
- Настройки сохраняются в `%APPDATA%\OverlayMusic\settings.json`

## Использование

| Действие | Hotkey |
|---|---|
| Показать / скрыть оверлей | `Ctrl+Alt+S` |
| Предыдущий трек (в фокусе оверлея) | `←` |
| Следующий трек (в фокусе оверлея) | `→` |
| Пауза / воспроизведение (в фокусе оверлея) | `↓` |
| Скрыть оверлей | `Esc` |

Хоткеи настраиваются: трей → правый клик → **Настройки**.

## Что работает

- ✅ Любой плеер с поддержкой SMTC: **Spotify, Yandex.Music, AIMP, foobar2000, Tidal, MusicBee, любой браузер с YouTube/SoundCloud**
- ✅ Cover-derived theming — обложка размывается в фон, прогресс-бар и play-кнопка красятся под акцент
- ✅ Discord Rich Presence с обложкой, временем, кнопкой «Открыть трек» в Я.Музыке (или другом сервисе на выбор)
- ✅ Прямые ссылки на трек в Яндекс.Музыке через их публичный API
- ✅ Запуск с правами администратора → работает поверх игр, запущенных от админа (CS, Valorant, etc.)
- ✅ Автозапуск без UAC-промпта при входе в Windows

## Известные ограничения

- **Яндекс.Музыка не репортит позицию в SMTC** — оверлей считает время локально от смены трека. Первый трек после запуска оверлея может показывать неточную позицию; со второго трека всё точно.
- В Discord **кнопка «Открыть трек» не показывается на твоём собственном профиле** — это политика Discord, она видна другим пользователям.
