# OverlayMusic v1.2.1 — Нативный drag + разгрузка NPSMSvc

## Что нового

- ⚡ **Перетаскивание окна теперь действительно плавное.** В v1.2.0 был исправлен лаг от пересчёта подложки, но окно всё равно провисало при быстрых движениях из-за того что Qt полностью перерисовывает frameless+translucent окно на каждый mouse-move. Теперь drag делегирован Windows через `startSystemMove()` — окном двигает оконный менеджер, Qt вообще не рендерит кадры во время перетаскивания.
- 🪶 **Разгрузка системной службы NPSMSvc** (Now Playing Session Manager). Раньше оверлей каждые 250 мс читал байты обложки трека через IPC к NPSMSvc — это давало службе 15-20% CPU и под 300 МБ памяти. Теперь:
  - Обложка тащится через IPC только при смене трека (раньше — каждый poll)
  - `MediaManager` кэшируется (раньше `request_async()` дёргался каждый poll)
  - Интервал poll-а 250 → 500 мс (прогресс-бар тикает локально через интерполяцию, на глаз не видно)

## Установка

### 🪟 Windows
- **`OverlayMusic-Setup-1.2.1.exe`** — установщик
- **`OverlayMusic.exe`** — портативный exe

### 🍏 macOS (Apple Silicon)
- **`OverlayMusic-arm64.dmg`** — перетащи в Applications
